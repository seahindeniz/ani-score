# {frontmatter.title}

AniScore is available as a browser extension for Chromium-based browsers.

## Chrome / Edge / Brave

Install AniScore from the [Chrome Web Store](https://chromewebstore.google.com/detail/aniscore/pjioojkbcigohkncmfoedipogbejlkko).

1. Visit the [Chrome Web Store listing](https://chromewebstore.google.com/detail/aniscore/pjioojkbcigohkncmfoedipogbejlkko)
2. Click **"Add to Chrome"**
3. Confirm the installation when prompted

The extension works on any Chromium-based browser including Google Chrome, Microsoft Edge, Brave, Opera, and Arc.

## Firefox

Firefox support is currently in development. Stay tuned for updates.

## After Installation

Once installed, navigate to a supported streaming site (e.g., [anizm.net](https://anizm.net)) and you'll start seeing AniList data on anime cards automatically.

### Connecting Your AniList Account

To see your personal watch progress and manage favorites:

1. Click the AniScore extension icon in your browser toolbar
2. Click **"Connect with AniList"**
3. Authorize AniScore on the AniList page
4. You're all set -- your watch progress will now appear on anime cards

### Customizing Colors

Open the AniScore side panel to customize genre and tag colors:

1. Right-click the AniScore extension icon
2. Select **"Open Side Panel"**
3. Use the color pickers to set your preferred colors for genres and tags
